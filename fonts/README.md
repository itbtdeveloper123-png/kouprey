Place Kantumruy Pro font files here to enable Khmer typography.

Expected filenames (used by the site):
- KantumruyPro-Regular.woff2
- KantumruyPro-Regular.woff
- KantumruyPro-Bold.woff2
- KantumruyPro-Bold.woff

If you have different filenames, either rename them to match the above, or update `app/Views/layout.php` @font-face src URLs accordingly.

If you don't have Kantumruy Pro or prefer a web-hosted font, you can remove the local @font-face lines and load a CDN font (e.g., Noto Sans Khmer) in the `<head>` instead.

Notes:
- Ensure licensing allows using Kantumruy Pro on your website.
- After dropping files here, clear browser cache and reload pages.
